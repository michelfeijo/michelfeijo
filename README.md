## 🏗️ Projetos em destaque

> Os projetos abaixo foram desenvolvidos para ambiente corporativo privado. O código-fonte pertence à empresa, mas os detalhes técnicos e funcionais estão documentados aqui.

---

### 🛒 Gestão de Categorias — E-commerce
Plataforma interna para gestão do catálogo de produtos do e-commerce do Grupo Koch.

- Gerenciamento hierárquico de **categorias de produtos**
- Upload e gestão de **imagens dos produtos**
- Cadastro de **endereço físico dentro da loja** (prédio e rua/corredor) para localização dos produtos
- Stack: **Next.js · TypeScript · Fastify · Oracle · Docker Swarm · GitHub Actions**

---

### 👷 Gestão de Promotores
Sistema web para controle de promotores de indústrias parceiras nas 94 lojas da rede.

- Check-in/check-out com saída automática após 8h
- Busca inteligente de fornecedores por nome, razão social e CNPJ
- Agenda diária por loja, avaliação de desempenho e solicitação de mão de obra com envio automático de e-mails
- Perfis diferenciados: Recepção, Gerente e Trade
- Stack: **React · TypeScript · Node.js · Oracle**

---

### 📊 Quadro Ideal — Planejamento de Força de Trabalho
Plataforma para controle do número ideal de colaboradores por cargo, loja, setor e período.

- Interface de planilha interativa (Handsontable) com duplicação de dados entre períodos e lojas
- Bloqueio de edição de meses passados
- Autenticação com perfis: Admin, Controladoria, RH e Diretoria
- Stack: **React · TypeScript · Chakra UI · Node.js · Oracle**

---

### 📋 Outros sistemas desenvolvidos
- **Plataforma de Convocação de Intermitentes** — convocação de trabalhadores intermitentes com Docker Swarm, Nginx e monitoramento via Grafana/Loki
- **Sistema de Gestão de Notas Fiscais** — fluxo de aprovação, reprova e integração com ERP Consinco
- **Integração de Assinaturas Digitais** — envio automático de contratos via ERP Sênior
- **Sistema de Cancelamento de Pedidos** — integração com Consinco e fluxo interno de aprovação
